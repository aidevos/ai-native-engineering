# 02 — Write for the AI that will read your work

Your code will be read by future AI agents, not just future humans. This changes what
good code looks like.

- Tests are now contracts that AI can check against.
- Comments are prompts for the next pass.
- Module interfaces are API contracts that agents query.
- Naming is no longer cosmetic — it is load-bearing for retrieval.

Engineer for this reader. It turns out it also makes the code better for humans.
