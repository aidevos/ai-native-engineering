# Experiments

Every experiment in this lab is a falsifiable probe of one hypothesis. Each has a success
criterion, a failure criterion, and a 7-day signal window. **No feedback is also feedback** —
we record silence explicitly.

We do not delete failed experiments. We do not retroactively edit conclusions.
